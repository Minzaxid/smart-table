# Умная таблица 1.0

Учебный проект из курса **Яндекс Практикум**. 
Это интерактивная таблица для вывода и обработки данных о продажах. Главная техническая особенность проекта — реализация сложной логики на чистом JavaScript (Vanilla JS) с использованием модульной архитектуры. 

Код разделен на независимые модули, которые последовательно обрабатывают данные, что делает проект расширяемым и удобным для поддержки.

## Что умеет страница

- **Пагинация:** разбивка большого массива данных на страницы с возможностью настройки количества отображаемых строк (10, 25, 50, 100).
- **Сортировка:** упорядочивание данных в колонках по возрастанию или убыванию.
- **Фильтрация:** поиск точных совпадений или фильтрация данных по конкретным полям (например, выбор продавца из выпадающего списка).
- **Глобальный поиск:** умный поиск введенного текста сразу по нескольким полям таблицы (дата, покупатель, продавец).
- **Сброс настроек:** возможность быстро сбросить все примененные фильтры одной кнопкой.

## Технологии

- HTML5 (использование тега `<template>` для клонирования элементов DOM).
- CSS3.
- JavaScript (ES6+ модули, стрелочные функции, методы массивов `map`, `filter`, `slice`, замыкания).
- Делегирование событий и работа с формами (`FormData`).
- Сборщик модулей **Vite**.
